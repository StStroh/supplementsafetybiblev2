### Contents

This archive contains all essential source code and configuration files for the Supplement Safety Bible application.

#### Included:

### Quick Start

1. **Extract the package:**
   ```bash
   tar -xzf complete-source-code.tar.gz
   cd project/
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   ```bash
   cp .env.example .env
   # Edit .env with your Supabase and Stripe keys
   ```

4. **Run locally:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```

### Environment Variables Required

Create a `.env` file with:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_STRIPE_PUBLISHABLE_KEY=your_stripe_key
VITE_SITE_URL=https://yourdomain.com
STRIPE_SECRET_KEY=your_stripe_secret_key
STRIPE_WEBHOOK_SECRET=your_webhook_secret
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```

### Database Setup

1. Create a Supabase project
2. Run all migrations in `supabase/migrations/` in order
3. Import seed data if needed
