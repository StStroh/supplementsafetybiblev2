## CTA Integration

Each page includes a prominent call-to-action:

**Button Text:**
"Check [Supplement] + [Medication]"

**Link Example:**
`/check?query=vitamin%20k%20warfarin`

**Behavior:**
- Opens checker with query hint
- Works for logged-out users
- Defaults to free tier if profile fails
- Pre-fills search context

## Content Template Structure

Each page follows this mandatory structure:

1. **H1:** Question-based headline
2. **Risk Summary:** 2-3 sentences above fold
3. **Why This Occurs:** Mechanism explanation
4. **Who Is At Risk:** Specific populations
5. **What to Do:** Actionable guidance
6. **CTA:** Checker integration with query params
7. **FAQ Section:** 2-4 questions with schema
8. **Related Links:** Internal navigation
